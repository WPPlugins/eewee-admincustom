== Description ==

= Management of admin custom, change admin logo wordpress by your logo, ... =
* Use a custom favicon on wordpress.
* Change the logo of the login page of the wordpress admin.
* Change description located in the footer of the wordpress admin
* Add custom notifications
* Add widgets to the dashboard of wordpress admin.
* Remove widgets to the dashboard of wordpress admin.
* Add help tab.
* Add background on login page.
* Admin admin bar.
* Add/Edit/Delete role.
* Add/Edit/Delete capabilities.
* Add breadcrumb

PS: question, [eewee.fr](http://www.eewee.fr).

* See the [Changelog](http://wordpress.org/extend/plugins/eewee-admincustom/changelog/) for what's new.

== Installation ==

1. Download the plugin Zip archive.
1. Upload plugin folder to your '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
